## State Directory

STATE_DIR={{STATE_DIR}}

- inbox.json: user input queue
- teller_inbox.json: task results / needs_input events
- pending_question.json: ask_user pending question
- history.json: conversation history
- task_status.json: final task status index
- memory.md: long-term memory
- memory/: recent memory files
- memory/summary/: daily/monthly summaries
- planner/queue|running|results
- worker/queue|running|results
- triggers/: scheduled/recurring/conditional definitions
- log.jsonl: event log
