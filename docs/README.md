# 文档导航

## 阅读入口

- 设计与运行机制：`./design/README.md`
- SDK 与外部对比：`./reference/README.md`
- 历史记录：`./archive/README.md`

## 目录约定

- `design/`：当前实现的单一事实源（架构、协议、接口、UI 规范）。
- `reference/`：外部 SDK 能力与技术选型材料。
- `archive/`：已完成或阶段性历史记录，不参与日常实现决策。

## 维护规则

- 同一事实只保留一个主文档，其他文档使用链接引用。
- 新增文档优先放入 `design/` 或 `reference/`；无持续维护价值的内容直接入 `archive/`。
- 调整路径后必须同步修复相对链接。
